## 🚀 Featured Project

<div align="center">

### 🏠 FlatMate — Smart Apartment Management Dashboard

*Built because shared apartments deserve smarter tools.*

</div>

<table>
<tr>
<td width="50%">

**✨ What it does**
- 📊 Real-time expense tracking with bar chart analytics
- ✅ Shared task checklist for all roommates
- 🧹 Weekly cleaning rotation schedule
- 📢 Apartment-wide announcements board
- 💱 Live currency rates & built-in converter
- 🤖 Telegram bot companion for quick updates
- 🌙 Fully dark-themed — easy on the eyes at 2AM

</td>
<td width="50%">

**🔧 Stack at a glance**

| Layer | Tech |
|---|---|
| Framework | Next.js 16 + TypeScript |
| Styling | Tailwind CSS v4 |
| Animation | Framer Motion |
| Database | Firebase Firestore |
| Sync | Real-time `onSnapshot` |
| Bot | Telegram Bot API |
| Theme | Dark · `#1D9E75` accent |

</td>
</tr>
</table>
