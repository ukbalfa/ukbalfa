<h2 align="center">/// DEPLOYMENTS</h2>

### 🏠 FlatMate [^Active]

**Architecture:** `Next.js 16` `·` `TypeScript` `·` `Firebase Firestore` `·` `Telegram Bot API`

A real-time, highly optimized management system bridging renters and landlords. Built for production-level roommate operations.
* **Telemetry & Finance:** Live expense tracking with bar chart analytics and live currency conversion.
* **Ops Automation:** Automated task checklists, weekly cleaning rotations, and synchronized announcements boards.
* **Integration:** Full companion Telegram bot for CLI-like command access on the go.
